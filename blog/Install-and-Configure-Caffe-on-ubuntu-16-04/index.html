Guide

requirements:

  • NVIDIA driver 396.54
  • CUDA 8.0 + cudnn 6.0.21
  • CUDA 9.2 +cudnn 7.1.4
  • opencv 3.1.0 —>3.3.0
  • python 2.7 + numpy 1.15.1
  • python 3.5.2 + numpy 1.16.2
  • protobuf 3.6.1 （static）
  • caffe latest

默认的protobuf,2.6.1测试通过。
此处，使用最新的3.6.1 也可以，编译caffe需要加上-std=c++11
